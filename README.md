# My 1st app created using Express.js

### Getting Started

-  Clone the repository
      ```sh
          $ https://github.com/marychellecarreon/expressjs_practice.git
     ```
     
-  Run npm install
      ```sh
          npm install
     ```
     
 -  Start the app. 
      ```sh
          node server.js
     ```
- Open browser and goto http://localhost:3000
